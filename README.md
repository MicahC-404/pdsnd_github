### Project Title
US Bikeshare Data Analysis

### Description
Project Overview: This script explores data on bike-sharing systems for three major cities: Chicago, New York City, and Washington. It provides an interactive terminal experience where users can filter data by city, month, and day to see descriptive statistics.

Key Features:
Interactive Filtering: Users choose the city and time period for analysis.

Time Statistics: Analyzes peak travel months, days, and hours.

Station Analysis: Identifies the most popular start/end stations and trips.

User Insights: Breaks down data by user type and, where available, gender and birth year.

Raw Data Browser: Allows users to scroll through the filtered dataset in 5-row increments.

How to Run:
Ensure Python, pandas, and numpy are installed.
Place the .py script and the three .csv files in the same directory.
Execute: python bikeshare.py
